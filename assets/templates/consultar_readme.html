Consultar agenda pacientes

Esta aplicación permite buscar rápidamente la información de un paciente y su agenda utilizando el RUT. Su objetivo es facilitar la validación de datos desde una sola pantalla, de forma simple, rápida y visual.

¿Qué hace esta aplicación?

Consulta la información asociada a un paciente ingresando su RUT y muestra los resultados en dos bloques principales:

  • Paciente: nombre, RUT, teléfono y correo.
  • Agenda: fecha, hora y estado de la cita.

¿Qué necesito para usarla?

  • Tener la aplicación abierta.
  • Ingresar un RUT válido en el campo de búsqueda.
  • Contar con conexión disponible para realizar la consulta.
La consulta se realiza automáticamente al presionar Enter o el botón Enviar.

Cómo usar la aplicación

  1. Abra la aplicación. Al iniciar, el cursor quedará listo en el campo de ingreso del RUT.
  2. Ingrese el RUT del paciente. Puede escribirlo y luego presionar Enter o hacer clic en Enviar.
  3. Espere la consulta. El sistema realizará la búsqueda y mostrará un indicador mientras obtiene la información.
  4. Revise el resultado. Si la consulta es exitosa, los campos de paciente y agenda se completarán automáticamente.

Bloque Paciente

En esta sección se mostrará la información principal del paciente consultado:

  • Nombre: nombre completo del paciente.
  • RUT: RUT formateado para una mejor lectura.
  • Teléfono: número con formato nacional.
  • Correo: correo electrónico en minúsculas.

Bloque Agenda

En esta sección se mostrará la información de la agenda asociada al paciente:

  • Fecha: fecha de la cita.
  • Hora: hora de atención.
  • Estado: estado actual de la agenda.

Formato esperado del RUT

El sistema normaliza el RUT para enviar la consulta en el formato requerido por la API. El formato final utilizado es:

Sin puntos, con guion y dígito verificador
Ejemplo: 20750523-4

Ejemplos de entrada válidos

  • 20.750.523-4
  • 20750523-4
  • 207505234
La aplicación limpia el formato del RUT antes de consultar. Si el formato ingresado es excesivamente largo o inválido, se mostrará un mensaje de error.

Mensajes y resultados

  • Consulta exitosa: se cargan los datos del paciente y agenda.
  • Paciente no encontrado: se muestra una ventana de error indicando que no existe información asociada al RUT ingresado.
  • RUT requerido: se muestra si intenta consultar sin ingresar datos.

Botón de ayuda

Junto al encabezado de la aplicación puede existir un botón de ayuda. Al presionarlo, se abrirá esta guía en formato HTML para que el usuario pueda revisar el funcionamiento del sistema en cualquier momento.

Recomendaciones de uso

  • Ingrese el RUT completo para evitar errores de búsqueda.
  • Verifique que exista conexión antes de realizar la consulta.
  • Espere a que termine la búsqueda antes de ingresar un nuevo RUT.
  • Si la información no aparece, revise el RUT ingresado y vuelva a consultar.
Si el sistema presenta un mensaje de error, revise el dato ingresado o contacte al área responsable si el problema persiste.

Resumen rápido

  1. Ingrese el RUT del paciente.
  2. Presione Enter o Enviar.
  3. Espere la respuesta.
  4. Revise los bloques Paciente y Agenda.
